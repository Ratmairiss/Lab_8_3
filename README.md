# Телефонная книга (Qt)

Десктопное приложение для управления контактами на C++ с применением фреймоврка Qt. Делалось как лабораторная работа в рамках курса по программитрованию на С++. Дополнительной задачей для реализации является хранение данных с применением алгоритма Хаффмана.

## Что умеет

- Добавление, редактирование и удаление контактов
- Поиск по любому полю в реальном времени
- Сортировка по имени, фамилии, отчеству, телефону, email, дате рождения (в обе стороны)
- Фото профиля (можно выбрать свой файл)
- Поле с произвольной информацией о контакте
- Данные сохраняются автоматически при закрытии

## Хранение данных

Контакты хранятся в двух форматах одновременно:

- `PersonsData.txt` — обычный текстовый файл, по 8 строк на контакт
- `PersonsDataHuffman.txt` + `CodeTablet.txt` — те же данные, сжатые алгоритмом Хаффмана

При запуске данные загружаются из обоих файлов, при закрытии — перезаписываются в оба.

## Сборка

Проект открывается в Visual Studio с установленным Qt (тестировалось на Qt 6). Нужно добавить путь к Qt в переменную `QTDIR` или настроить Qt VS Tools.

```
Открыть Lab_8_3.sln → Build → Run
```

## Структура проекта

```
main.cpp             — точка входа, загрузка стиля
PhoneDirectory.h/cpp — главный виджет, вся логика приложения
Person.h/cpp         — структура контакта и компараторы для сортировки
InputTab.h           — вкладка с полями ввода (используется и для поиска, и для добавления)
SpecialButton.h      — кнопка контакта в списке (имя + кнопка удаления)
style.qss            — стили приложения
Resource.qrc         — ресурсы (стиль, иконки)
PersonsData.txt      — база контактов в текстовом виде
PersonsDataHuffman.txt / CodeTablet.txt — сжатая версия базы
```
